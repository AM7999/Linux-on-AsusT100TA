Installing 64 bit Linux on the Asus T100 TA

A Complete and comprehensive guide to installing 64 bit linux on the ASUS T100TA

So Why this guide if its super easy to install linux on pretty much everything?

This is because the Asus T100 TA, while has a 64 bit CPU, does not have a 64 bit UEFI meaning it can only boot 32 bit EFI files
